DreamMusic
DreamMusic is a React-based music streaming application interface. It provides a sleek, modern design inspired by popular music streaming platforms, featuring a dark theme with red accents.

Project Overview
DreamMusic is a front-end application that showcases:

A responsive layout with a sidebar, main content area, and music player
An artist page display with a banner and list of popular songs
A search functionality in the header
A persistent music player at the bottom of the interface
Technologies Used
React
TypeScript
Tailwind CSS for styling
Lucide React for icons

Installation
Clone the repository:

git clone https://github.com/yourusername/dreammusic.git
cd dreammusic
Install the dependencies:

npm install
Start the development server:

npm start
Open http://localhost:3000 in your browser to view the application.

Available Scripts
In the project directory, you can run:

npm start: Runs the app in development mode.
npm test: Launches the test runner.
npm run build: Builds the app for production to the build folder.
npm run eject: Removes the single build dependency from your project.
